# Continuous Integration (CI) 🏃🏻‍♂️

- Jenkins 👨🏻‍🍳
- You project context and infra used

## Key practices

- Maximize your tests value by setting up your tests in CI **from the start**
- Setup smoke tests for every build
- Setup regression on a nightly basis

> Tip: 💡Make sure the suite is always healthy 🏥with less false positives/negatives

## Available choices

- Azure pipelines, GitLab, TravisCI, SemaphoreCI, CircleCI, Bamboo, Code-ship ...
